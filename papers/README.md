# Conference Papers

Each venue folder contains a distinct manuscript, bibliography, figures, and
submission materials. Shared implementation belongs in `src/metarouter` or
`src/coageneration`; paper folders must not contain copied source trees.

- `dai2026/`: MetaRoute-Bench framework, artifact, operational evaluation, and deployment lessons
- `aaai2027/`: learned budget-aware routing method and executable-task evaluation
- `oracle2026/`: multilingual and culturally grounded routing evaluation
- `coa-bench/`: early-stage draft evaluating COA generation policies via self-play (venue not yet selected; see `coa-bench/README.md`)

The papers may share infrastructure and background citations, but each archival
submission must have a distinct research question, method, primary experiments,
headline results, and conclusion.
