# ORACLE 2026 Workshop Paper

## Intended Contribution

An evaluation of whether agentic routing policies transfer across languages,
scripts, code-switching, resource levels, and culturally grounded tasks.

## Required Evidence

- Parallel and culturally grounded multilingual tasks
- Typologically and script-diverse languages
- English-only, translated, language-aware, and multilingual routing baselines
- Routing consistency, success, cost, latency, and calibration by language
- Qualitative analysis with native-speaker or qualified linguistic review

Use the workshop's non-archival option if the final contribution substantially
overlaps another archival submission.
