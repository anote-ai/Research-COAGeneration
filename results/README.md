# Experiment Artifacts

`dai2026/main/` contains the eight-policy DAI experiment: 180 generated task
profiles, 30 paired seeds, and 43,200 traces. `dai2026/ablations/` contains five
adaptive-policy conditions and 27,000 traces. AAAI and ORACLE results will be
stored in their own venue directories and must come from distinct experiments.

Files:

- `tasks.csv`: benchmark task profiles and routing-need annotations
- `traces.csv`: one row per task, policy, and seed execution
- `summary.csv`: aggregate success, cost, latency, and utility
- `comparisons.csv`: paired differences between adaptive and comparison policies
- `details.json`: configuration, workload success, action counts, and failures

All outcomes are generated by the offline simulator in `src/metarouter/simulator.py`.
They are not measurements from a production deployment or live language model.
